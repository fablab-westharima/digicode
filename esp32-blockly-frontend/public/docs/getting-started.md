# はじめかた - DigiCode

DigiCodeは、ESP32マイコンを使ったロボット・IoTデバイスのプログラミングを、ブロックで簡単に行えるビジュアル開発環境です。

## 必要なもの

### ハードウェア
- **ESP32開発ボード** - [推奨ハードウェアリスト](./recommended-hardware.md)を参照
- USBケーブル（Type-CまたはMicro-USB、ボードにより異なります）
- センサーやモーター（作りたいものに応じて）

> **重要:** 安定した動作のため、[推奨ハードウェアリスト](./recommended-hardware.md)に掲載されている販売店からの購入を推奨します。

### ソフトウェア
- 最新のWebブラウザ（Chrome、Edge、Safari推奨）
- インターネット接続

## ステップ1: アカウント作成とログイン

1. DigiCodeにアクセス
2. 「新規登録」をクリック
3. メールアドレスとパスワードを入力
4. 登録完了後、自動的にログイン

## ステップ2: ESP32にファームウェアを書き込む

DigiCodeを使用するには、最初にESP32に専用ファームウェアを書き込む必要があります。

1. ホームページで「**ファームウェア書き込み**」をクリック
2. ESP32をUSBケーブルでPCに接続
3. 「DigiCode競技用（Arduino C++）」を選択
4. 「INSTALL」ボタンをクリック
5. 接続するシリアルポートを選択
6. 書き込みが完了するまで待つ（約1分）

## ステップ3: 最初のプログラム（LED点滅）

### プログラムを作成

1. ホームページで「**新規プロジェクト**」をクリック
2. Blocklyエディタが開きます
3. 左側のツールボックスから以下のブロックを選択：

#### セットアップブロック
```
セットアップ:
  ピンモード設定 [2] を [出力]
```

#### ループブロック
```
ずっと繰り返す:
  デジタル書き込み [2] を [HIGH]
  待機 [1000] ミリ秒
  デジタル書き込み [2] を [LOW]
  待機 [1000] ミリ秒
```

### ESP32に書き込む

1. 「**書き込み**」ボタンをクリック
2. 書き込み方法を選択：
   - **USB接続で書き込み** - 初回はこちら
   - WiFi OTA書き込み - 2回目以降、WiFi設定済みの場合
3. シリアルポートを選択
4. 書き込み完了を待つ

### 動作確認

ESP32の内蔵LED（GPIO2）が1秒間隔で点滅します！

## ステップ4: サンプルプロジェクトを試す

1. エディタで「サンプル」ボタンをクリック
2. カテゴリから選択：
   - **基本** - LED点滅、シリアル通信
   - **センサー** - 超音波センサー、温湿度センサー
   - **競技** - ライントレース、マイクロマウス
3. サンプルを選択して読み込み
4. 必要に応じてピン番号を変更
5. 書き込んで動作確認
