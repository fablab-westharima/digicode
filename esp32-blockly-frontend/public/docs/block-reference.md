## 基本ブロック

プログラムの基本構造とESP32の入出力を制御するブロックです。

### プログラム構造

| ブロック名 | 説明 | 備考 |
|-----------|------|------|
| **セットアップ** | プログラム開始時に1回だけ実行される | 初期化処理を配置 |
| **ループ（繰り返し）** | 繰り返し実行されるメイン処理 | センサー読み取りや制御処理を配置 |

### デジタル入出力

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **ピンモード設定** | ピンの入出力モードを設定 | ピン番号(0-39)、モード(INPUT/OUTPUT/INPUT_PULLUP) |
| **デジタル出力** | ピンにHIGH/LOWを出力 | ピン番号、値(HIGH/LOW) |
| **デジタル入力** | ピンの状態を読み取る | ピン番号 → true/false |
| **内蔵LED ON** | ESP32内蔵LED(GPIO2)を点灯 | - |
| **内蔵LED OFF** | ESP32内蔵LED(GPIO2)を消灯 | - |

### アナログ入出力

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **アナログ入力** | アナログ値を読み取る(0-4095) | ピン番号(GPIO32-39推奨) |
| **PWM出力** | PWM信号を出力 | ピン番号、値(0-255) |

### 時間・タイミング

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **待つ** | 指定時間待機 | 時間(ミリ秒) |
| **経過時間** | 起動からの経過時間を取得 | → ミリ秒 |

### シリアル通信

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **シリアル通信開始** | シリアル通信を初期化 | ボーレート(9600/57600/115200) |
| **シリアル出力** | シリアルモニタに出力 | 出力値 |
| **シリアル出力(改行あり)** | 改行付きでシリアル出力 | 出力値 |

### その他

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **音を鳴らす** | 指定周波数の音を出力 | ピン番号、周波数(Hz) |

## データ（配列）

配列を操作するブロックです。

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **配列を作成** | 1次元/2次元/3次元配列を作成 | 変数名、型、次元、サイズ |
| **配列要素設定** | 配列の要素に値を設定 | 変数名、インデックス、値 |
| **配列要素取得** | 配列の要素を取得 | 変数名、インデックス → 値 |
| **配列サイズ取得** | 配列の要素数を取得 | 変数名 → 数値 |
| **配列内容** | 配列の初期値を定義 | 要素数、各要素の値 |

## センサー

### 超音波センサー（HC-SR04）

距離を測定するセンサーです。

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **HC-SR04初期化** | 超音波センサーを初期化 | Trigピン、Echoピン |
| **超音波距離(cm)** | 距離を測定 | → cm単位の数値 |

**接続例:**
- Trig → GPIO18
- Echo → GPIO19
- VCC → 5V
- GND → GND

### RUS-04（RGB付き超音波センサー）

OTTOロボット用のRGB LED内蔵超音波センサーです。

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **RUS-04初期化** | センサーとRGB LEDを初期化 | Trigピン、Echoピン、RGBピン、LED数 |
| **RUS-04距離(cm)** | 距離を測定 | → cm単位の数値 |
| **RUS-04 RGB両目** | 両目の色を設定 | 左色、右色 |
| **RUS-04 RGB 目** | 片目の色を設定 | 左/右、色 |
| **RUS-04 RGB 明るさ** | LEDの明るさを設定 | 0-255 |
| **RUS-04 RGB 消灯** | LEDを消灯 | - |

### 温湿度センサー（DHT11/DHT22）

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **DHT初期化** | DHTセンサーを初期化 | タイプ(DHT11/DHT22)、ピン |
| **DHT温度(℃)** | 温度を取得 | → 摂氏温度 |
| **DHT湿度(%)** | 湿度を取得 | → パーセント |

### タッチセンサー

ESP32内蔵の静電容量式タッチセンサーです。

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **タッチセンサー初期化** | タッチセンサーを初期化 | ピン(T0-T9)、しきい値 |
| **タッチされた？** | タッチ検出 | → true/false |
| **タッチセンサー値** | 生の値を取得 | → 数値(小さいほど強くタッチ) |

**対応ピン:** T0(GPIO4)、T2(GPIO2)、T3(GPIO15)など

### サウンドセンサー

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **サウンドセンサー初期化** | マイクセンサーを初期化 | ADCピン(GPIO32-39) |
| **音量レベル** | 音量を取得 | → 0-4095 |
| **音が聞こえた？** | しきい値以上の音を検出 | しきい値 → true/false |

### 光センサー

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **光センサー初期化** | CdSセンサーを初期化 | ADCピン(GPIO32-39) |
| **光量レベル** | 明るさを取得 | → 0-4095(大きいほど明るい) |

### デジタルセンサー

| ブロック名 | 説明 | 検出対象 |
|-----------|------|----------|
| **ボタン** | ボタン/スイッチ | 押下状態 |
| **人感センサー(PIR)** | 赤外線人感センサー | 動き |
| **傾斜センサー** | 傾きセンサー | 傾き |
| **振動センサー** | 振動/衝撃センサー | 振動 |
| **磁気センサー** | ホール効果センサー | 磁石 |
| **フォトインタラプタ** | 赤外線遮断センサー | 遮断 |
| **IR障害物センサー** | 赤外線障害物センサー | 障害物 |
| **炎センサー** | 炎検出センサー | 火炎 |
| **ガスセンサー** | MQシリーズガスセンサー | ガス |
| **リミットスイッチ** | マイクロスイッチ | 接触 |

### アナログセンサー

| ブロック名 | 説明 | 出力モード |
|-----------|------|-----------|
| **ポテンショメータ** | 可変抵抗器 | 生値/パーセント/角度 |
| **光センサー(LDR)** | CdS光センサー | 生値/パーセント |
| **サーミスタ温度** | NTCサーミスタ | 摂氏温度 |
| **LM35温度** | LM35温度センサー | 摂氏温度 |
| **土壌水分** | 土壌水分センサー | 生値/パーセント |
| **水位センサー** | 水位検出センサー | 生値/パーセント |
| **ジョイスティック** | 2軸アナログスティック | X軸/Y軸 |

### ラインセンサー

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **汎用ラインセンサー初期化** | QTR/TCRTセンサーを初期化 | センサー数(2-8)、各ピン |

## アクチュエーター

### サーボモーター

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **サーボ接続** | サーボをピンに接続 | ピン番号 |
| **サーボ角度** | サーボを指定角度に回転 | ピン、角度(0-180°) |
| **サーボ角度(変数)** | 変数で角度を指定 | ピン、角度値 |
| **サーボ切断** | サーボを切断(省電力) | ピン |
| **サーボスイープ** | 開始から終了角度まで移動 | ピン、開始角度、終了角度、速度 |

### DCモーター（L298N）

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **モーター初期化** | L298Nモータードライバーを初期化 | モーター(A/B)、IN1/IN2/ENAピン |
| **モーター動作** | モーターを駆動 | モーター、方向(前進/後退/停止)、速度(0-255) |
| **モーター速度** | 速度のみ変更 | モーター、速度 |
| **モーター停止** | モーターを停止 | モーター(A/B/両方) |

### ステッピングモーター

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **ステッパー初期化** | 28BYJ-48を初期化 | IN1-IN4ピン |
| **ステッパー回転** | ステップ数分回転 | ステップ数、方向、速度 |

## NeoPixel（LED）

WS2812B等のアドレッサブルLEDを制御します。

### 基本操作

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **NeoPixelリング初期化** | LEDストリップを初期化 | ピン、LED数 |
| **LED色設定** | 個別LEDの色を設定 | インデックス、R/G/B(0-255) |
| **全LED色設定** | 全LEDを同じ色に | R/G/B(0-255) |
| **NeoPixel表示更新** | 色を反映 | - |
| **NeoPixel全消灯** | 全LEDを消灯 | - |
| **NeoPixel明るさ設定** | 全体の明るさ | 0-255 |

### エフェクト

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **レインボーエフェクト** | 虹色アニメーション | 速度 |
| **バウンスエフェクト** | 跳ねるLED | 色、速度 |
| **サイクルエフェクト** | 回転エフェクト | 色、速度 |

### シンプル操作

| ブロック名 | 説明 | パラメータ |
|-----------|------|-----------|
| **全LED色（シンプル）** | 色名で全LED設定 | 色(赤/緑/青/黄/紫/水色/白/消灯) |
| **LED # 色（シンプル）** | 色名で個別LED設定 | インデックス、色 |

## OTTOロボット

| **OTTO Ninja動作** | 動作実行 | 動作種類 |

## 対応言語

すべてのブロックは以下の言語に対応しています：

- **Arduino C++** - ESP32 Arduino フレームワーク
- **MicroPython** - ESP32 MicroPython 環境

ブロックを配置すると、選択した言語に対応するコードが自動生成されます。
