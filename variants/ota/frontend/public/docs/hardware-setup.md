# ハードウェア接続ガイド

ESP32とセンサー・アクチュエーターの接続方法を説明します。

> **推奨:** 使用するデバイスは[推奨ハードウェアリスト](./recommended-hardware.md)を参照してください。動作確認済みの部品を使用することで、トラブルを防げます。

## ESP32ピン配置

ESP32開発ボードは様々な種類がありますが、一般的なピン配置は以下の通りです：

### GPIO（汎用入出力）
- **GPIO 0-39**: デジタル入出力
- **GPIO 32-39**: アナログ入力専用（ADC1）
- **GPIO 0, 2**: 内蔵LED（ボードによる）

### 特殊ピン
- **GPIO 1, 3**: UART（シリアル通信）- プログラム書き込みに使用されるため避ける
- **GPIO 6-11**: フラッシュメモリ専用 - 使用不可
- **GPIO 34-39**: 入力専用（プルアップ抵抗なし）

### 電源
- **3.3V**: センサー・モーター用電源
- **5V**: USB経由の5V電源（一部のボードのみ）
- **GND**: グラウンド（共通接地）

## センサー接続

### 超音波距離センサー（HC-SR04）

**ピン配置:**
```
HC-SR04    ESP32
--------   ------
VCC    →   3.3V or 5V
GND    →   GND
Trig   →   GPIO 5
Echo   →   GPIO 18
```

**注意:** 5V駆動のセンサーをESP32（3.3V）に接続する場合、Echoピンに分圧回路を入れる

### 温湿度センサー（DHT11/DHT22）

**ピン配置:**
```
DHT11      ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
DATA   →   GPIO 4
```

**注意:** DATAピンとVCCの間に10kΩプルアップ抵抗を接続

### QTRラインセンサー（8ch）

**ピン配置例:**
```
QTR-8A     ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
1      →   GPIO 36
2      →   GPIO 39
3      →   GPIO 34
4      →   GPIO 35
5      →   GPIO 32
6      →   GPIO 33
7      →   GPIO 25
8      →   GPIO 26
```

## アクチュエーター接続

### サーボモーター（SG90等）

**ピン配置:**
```
Servo      ESP32
--------   ------
VCC    →   5V（外部電源推奨）
GND    →   GND
Signal →   GPIO 13
```

**注意:**
- 複数のサーボを使用する場合は、外部5V電源を使用
- ESP32のGNDと外部電源のGNDを共通接地

### DCモーター（モータードライバ経由）

**L298Nモータードライバー使用例:**
```
L298N      ESP32
--------   ------
IN1    →   GPIO 16
IN2    →   GPIO 17
ENA    →   GPIO 25（PWM）
GND    →   GND
```

**電源:**
- L298NのVCCには外部電源（7-12V）を接続
- L298NのGNDとESP32のGNDを共通接地

### NeoPixel LED（WS2812B）

**ピン配置:**
```
NeoPixel   ESP32
--------   ------
VCC    →   5V
GND    →   GND
DIN    →   GPIO 23
```

**注意:** 多数のLEDを使用する場合は外部5V電源を推奨

## 配線のベストプラクティス

### 1. 電源管理
- センサーは3.3V、モーターは外部電源を使用
- GNDは必ず共通接地
- 電源容量を確認（ESP32の3.3Vピンは最大200mA程度）

### 2. ノイズ対策
- モーターとESP32の間に距離を置く
- モーター電源にコンデンサを入れる
- 配線をできるだけ短く

### 3. 安全対策
- ショート防止のため、配線を確認してから電源を入れる
- 過電流防止のため、適切なヒューズや保護回路を使用
- 高電圧（12V以上）を扱う場合は特に注意

## トラブルシューティング

### センサーが動作しない
1. 配線（VCC、GND、信号）を再確認
2. 電圧レベルを確認（3.3V/5V）
3. プルアップ/プルダウン抵抗の有無を確認

### モーターが動かない
1. 外部電源が供給されているか確認
2. GNDの共通接地を確認
3. モータードライバーの配線を確認
