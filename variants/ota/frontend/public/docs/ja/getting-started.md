# はじめかた - DigiCode

**最終更新:** 2025-12-28

DigiCodeは、ESP32マイコンを使ったロボット・IoTデバイスのプログラミングを、ブロックで簡単に行えるビジュアル開発環境です。

![セットアップ4ステップ](/docs/images/getting-started-steps.svg)

## 必要なもの

### ハードウェア
- **ESP32開発ボード** - [推奨ハードウェアリスト](./recommended-hardware.md)を参照
- USBケーブル（Type-CまたはMicro-USB、ボードにより異なります）
- センサーやモーター（作りたいものに応じて）

> **重要:** 安定した動作のため、[推奨ハードウェアリスト](./recommended-hardware.md)に掲載されている販売店からの購入を推奨します。

### ソフトウェア
- 最新のWebブラウザ（Chrome、Edge推奨）
- インターネット接続
- **DigiCode Finder**（WiFi OTA書き込み用デスクトップアプリ、任意）

## ステップ1: アカウント作成とログイン

1. DigiCodeにアクセス
2. 「新規登録」をクリック
3. メールアドレスとパスワードを入力
4. 登録完了後、自動的にログイン

## ステップ2: ESP32にファームウェアを書き込む

DigiCodeを使用するには、最初にESP32に専用ファームウェアを書き込む必要があります。

1. 左メニューの「**ファームウェア書き込み**」をクリック
2. ESP32をUSBケーブルでPCに接続
3. 「INSTALL」ボタンをクリック
4. 接続するシリアルポートを選択
5. 書き込みが完了するまで待つ（約1分）

> **注意:** ファームウェアは一度書き込めば、通常は再書き込み不要です。プログラムは何度でも書き換えできます。

## ステップ3: 最初のプログラム（LED点滅）

### プログラムを作成

1. エディタ画面が表示されます
2. 左側のツールボックスから以下のブロックを選択：

#### セットアップブロック
```
セットアップ:
  ピンモード設定 [2] を [出力]
```

#### ループブロック
```
ずっと繰り返す:
  デジタル書き込み [2] を [HIGH]
  待機 [1000] ミリ秒
  デジタル書き込み [2] を [LOW]
  待機 [1000] ミリ秒
```

### ESP32に書き込む

1. 「**書き込み**」ボタンをクリック
2. 書き込み方法を選択：

| 方式 | 用途 | 備考 |
|------|------|------|
| **WiFi OTA** | 通常の更新（最速） | DigiCode Finderでデバイス検出 |
| **BLE** | ケーブルなし更新 | Web Bluetooth対応ブラウザ必要 |
| **USB** | 初回設定・確実な書き込み | ケーブル接続必要 |

#### WiFi OTA書き込みの場合（推奨）

1. **DigiCode Finder**（デスクトップアプリ）を起動
2. ネットワーク上のDigiCodeデバイスが自動検出される
3. 「選択」ボタンでIPアドレスをコピー
4. DigiCode（ブラウザ）に戻り「WiFi OTA」を選択
5. IPアドレスをペーストして書き込み開始

> **DigiCode Finderのダウンロード:**
> https://github.com/fablab-westharima/DigiCode-Finder/releases

#### USB書き込みの場合

1. 「USB」を選択
2. シリアルポートを選択
3. 書き込み完了を待つ

### 動作確認

ESP32の内蔵LED（GPIO2）が1秒間隔で点滅します！

## ステップ4: サンプルプロジェクトを試す

1. エディタで「サンプル」ボタンをクリック
2. カテゴリから選択：
   - **基本** - LED点滅、シリアル通信
   - **センサー** - 超音波センサー、温湿度センサー
   - **競技** - ライントレース、マイクロマウス
3. サンプルを選択して読み込み
4. 必要に応じてピン番号を変更
5. 書き込んで動作確認

## 3つの書き込み方式の比較

| 方式 | メリット | デメリット |
|------|----------|-----------|
| **WiFi OTA** | 高速、ケーブル不要 | 初回はファームウェア書込み必要 |
| **BLE** | ケーブル不要、AP制限環境でも使用可能 | 書き込み速度がやや遅い |
| **USB** | 確実、どの環境でも動作 | ケーブル接続が必要 |

**推奨:** 初回はUSBでファームウェアを書き込み、その後はWiFi OTAで更新
