## ESP32 ピン早見表

ESP32 開発ボードは様々ですが、共通の前提知識:

| ピン | 用途 | 注意 |
|---|---|---|
| **GPIO 0-39** | デジタル入出力 | — |
| **GPIO 32-39** | アナログ入力 (ADC1) のみ | ADC2 は WiFi 使用時に競合 |
| **GPIO 0, 2** | 内蔵 LED (ボード依存) | — |
| **GPIO 1, 3** | UART (シリアル通信) | プログラム書き込みに使用、避ける |
| **GPIO 6-11** | フラッシュメモリ専用 | 使用不可 |
| **GPIO 34-39** | 入力専用 | プルアップ抵抗なし |
| **3.3V** | センサー電源 | 最大 200mA |
| **5V** | USB 5V (一部ボード) | モーター用は外部電源推奨 |
| **GND** | グラウンド | 共通接地必須 |

> 💡 **I2C 標準ピン**: SDA = GPIO 21、SCL = GPIO 22 (ESP32 デフォルト)
> 💡 **SPI 標準ピン**: SCK = GPIO 18、MOSI = GPIO 23、MISO = GPIO 19 (ハードウェア SPI)

## センサー接続

### 超音波距離センサー (HC-SR04)

```
HC-SR04    ESP32
--------   ------
VCC    →   3.3V or 5V
GND    →   GND
Trig   →   GPIO 5
Echo   →   GPIO 18
```

**注意:** 5V 駆動のセンサーを ESP32 (3.3V) に接続する場合、Echo ピンに分圧回路を入れる。

### 温湿度センサー (DHT11 / DHT22)

```
DHT11      ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
DATA   →   GPIO 4
```

**注意:** DATA ピンと VCC の間に 10kΩ プルアップ抵抗を接続。

### QTR ラインセンサー (8ch)

```
QTR-8A     ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
1      →   GPIO 36
2      →   GPIO 39
3      →   GPIO 34
4      →   GPIO 35
5      →   GPIO 32
6      →   GPIO 33
7      →   GPIO 25
8      →   GPIO 26
```

### 加速度・ジャイロセンサー (MPU6050)

```
MPU6050    ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
SDA    →   GPIO 21
SCL    →   GPIO 22
```

**注意:** アドレスは 0x68 (AD0=LOW) または 0x69 (AD0=HIGH)。同一 I2C バスに複数台接続可。

### 気圧・温湿度センサー (BME280 / BMP280)

```
BME280     ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
SDA    →   GPIO 21
SCL    →   GPIO 22
```

**注意:** デフォルトアドレス 0x76 (SDO=LOW) または 0x77 (SDO=HIGH)。BMP280 は温度・気圧のみ、BME280 は湿度も計測可。

### ToF 距離センサー (VL53L0X)

```
VL53L0X    ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
SDA    →   GPIO 21
SCL    →   GPIO 22
```

**注意:** アドレス 0x29。複数台同時使用時は XSHUT ピンで個別に起動し、アドレスを変更する。

### 磁気エンコーダ (AS5600)

```
AS5600     ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
SDA    →   GPIO 21
SCL    →   GPIO 22
DIR    →   GND (時計回り正方向) または 3.3V (反時計回り正方向)
```

**注意:** アドレス 0x36 固定。マグネットはシャフト上面中央に取り付け、センサーとの距離は 0.5〜3mm。

## アクチュエーター接続

### サーボモーター (SG90 等)

```
Servo      ESP32
--------   ------
VCC    →   5V (外部電源推奨)
GND    →   GND
Signal →   GPIO 13
```

**注意:**
- 複数のサーボを使用する場合は、外部 5V 電源を使用
- ESP32 の GND と外部電源の GND を共通接地

### DC モーター (モータードライバ経由)

**L298N モータードライバー使用例:**

```
L298N      ESP32
--------   ------
IN1    →   GPIO 16
IN2    →   GPIO 17
ENA    →   GPIO 25 (PWM)
GND    →   GND
```

**電源:**
- L298N の VCC には外部電源 (7-12V) を接続
- L298N の GND と ESP32 の GND を共通接地

### NeoPixel LED (WS2812B)

```
NeoPixel   ESP32
--------   ------
VCC    →   5V
GND    →   GND
DIN    →   GPIO 23
```

**注意:** 多数の LED を使用する場合は外部 5V 電源を推奨。

## 通信モジュール接続

### I2C LCD ディスプレイ

PCF8574 I2C アダプタ付き 16x2 LCD。

```
I2C LCD    ESP32
--------   ------
VCC    →   5V (推奨) または 3.3V
GND    →   GND
SDA    →   GPIO 21
SCL    →   GPIO 22
```

**注意:** PCF8574 アダプタのジャンパで I2C アドレスを設定 (デフォルト 0x27)。LCD 電源は 5V 推奨 (3.3V では表示が暗い場合あり)。

### TFT ディスプレイ (SPI)

ST7789 / ILI9341 等、ハードウェア SPI 接続。

```
TFT        ESP32
--------   ------
VCC    →   3.3V
GND    →   GND
CS     →   GPIO 5
DC     →   GPIO 2
RST    →   GPIO 4
SCK    →   GPIO 18 (SPI CLK)
MOSI   →   GPIO 23 (SPI MOSI)
MISO   →   GPIO 19 (SPI MISO、省略可)
BL     →   3.3V またはバックライト制御ピン
```

**注意:** ドライバ IC (ST7789 / ILI9341 / ST7735) をブロックで選択する。RFID と SPI バスを共有する場合は CS ピンを変更すること。

### RFID リーダー (RC522)

ハードウェア SPI 接続。

```
RC522      ESP32
--------   ------
3.3V   →   3.3V
GND    →   GND
CS     →   GPIO 5
SCK    →   GPIO 18 (SPI CLK)
MOSI   →   GPIO 23 (SPI MOSI)
MISO   →   GPIO 19 (SPI MISO)
RST    →   GPIO 22
IRQ    →   未接続 (ポーリング方式では不要)
```

**注意:** RC522 は 3.3V 専用 (5V 接続禁止)。TFT と SPI バスを共有する場合は CS ピンを変更すること。電波法に関する注意は [推奨ハードウェアリスト](./recommended-hardware.md) を参照。

### IR 受信モジュール (VS1838B)

```
VS1838B    ESP32
--------   ------
VCC    →   3.3V または 5V
GND    →   GND
OUT    →   GPIO 14
```

**注意:** OUT ピンはモジュール側でプルアップ済み。IRremoteESP8266 ライブラリを使用 (ESP32 でも動作)。

### DFPlayer Mini (MP3 再生)

UART 接続。

```
DFPlayer   ESP32
--------   ------
VCC    →   5V
GND    →   GND
RX     →   GPIO 12 (1kΩ 直列抵抗経由)
TX     →   GPIO 14
SPK+   →   スピーカー (+)
SPK−   →   スピーカー (−)
```

**注意:** DFPlayer の RX ピンには 1kΩ 保護抵抗を必ず直列に入れる。SoftwareSerial 使用 (ESP32 RX=14、TX=12)。microSD カードには MP3 ファイルを `/01/001.mp3` 形式で配置。

## カメラモジュール接続

ESP32 カメラモジュールは専用ピン配置が組み込まれており、ボード種類をブロックで選択するだけで設定完了します。

### ESP32-CAM (AI-Thinker)

基板上にカメラコネクタが実装済み。別途 OV2640 カメラを接続します。

**書き込み時の注意:**
- IO0 と GND をジャンパで接続してから USB-UART アダプタで書き込む
- 書き込み後はジャンパを外して電源を入れ直す
- ESP32-CAM には USB ポートがないため、USB-UART アダプタ (3.3V) が必要

**電源:** 5V (USB 経由) または外部 5V。3.3V ピンへの直接給電はカメラ動作中の電流不足を起こしやすいため非推奨。

### XIAO ESP32S3 Sense

カメラ (OV2640) とマイクが基板に内蔵されています。専用フレキシブルケーブルでカメラを接続するだけで使用可能、外部への追加配線は不要です。

## CAN Bus 接続 (TWAI)

ESP32 内蔵 TWAI コントローラを使用。外部に CAN トランシーバ IC (SN65HVD230 等) が必要です。

```
SN65HVD230  ESP32
----------  ------
VCC     →   3.3V
GND     →   GND
D (TXD) →   GPIO 5 (TX)
R (RXD) →   GPIO 4 (RX)
CANH    →   CAN バスの CANH ライン
CANL    →   CAN バスの CANL ライン
```

**注意:** CAN バスの終端には 120Ω 終端抵抗を両端に接続する。SN65HVD230 は 3.3V 動作。電波法上の注意事項は [推奨ハードウェアリスト](./recommended-hardware.md) を参照。

## 詳細・運用

### 配線ベストプラクティス

#### 1. 電源管理

- センサーは 3.3V、モーターは外部電源を使用
- GND は必ず共通接地
- 電源容量を確認 (ESP32 の 3.3V ピンは最大 200mA 程度)

#### 2. ノイズ対策

- モーターと ESP32 の間に距離を置く
- モーター電源にコンデンサを入れる
- 配線をできるだけ短く

#### 3. 安全対策

- ショート防止のため、配線を確認してから電源を入れる
- 過電流防止のため、適切なヒューズや保護回路を使用
- 高電圧 (12V 以上) を扱う場合は特に注意

### トラブルシューティング

#### センサーが動作しない

1. 配線 (VCC、GND、信号) を再確認
2. 電圧レベルを確認 (3.3V / 5V)
3. プルアップ / プルダウン抵抗の有無を確認

#### モーターが動かない

1. 外部電源が供給されているか確認
2. GND の共通接地を確認
3. モータードライバーの配線を確認

詳細は [トラブルシューティング](./troubleshooting.md) を参照。
