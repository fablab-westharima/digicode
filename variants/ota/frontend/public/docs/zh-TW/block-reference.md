## 基礎積木

用於基本程式結構和 ESP32 I/O 控制的積木。

### 程式結構

| 積木名稱 | 說明 | 備註 |
|----------|------|------|
| **設定** | 程式啟動時執行一次 | 放置初始化程式碼 |
| **迴圈** | 重複執行的主要處理 | 放置感測器讀取和控制邏輯 |

### 數位 I/O

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **設定腳位模式** | 設定腳位輸入/輸出模式 | 腳位編號（0-39）、模式（INPUT/OUTPUT/INPUT_PULLUP） |
| **數位寫入** | 輸出 HIGH/LOW 到腳位 | 腳位編號、值（HIGH/LOW） |
| **數位讀取** | 讀取腳位狀態 | 腳位編號 → true/false |
| **內建 LED 開** | 開啟 ESP32 內建 LED（GPIO2） | - |
| **內建 LED 關** | 關閉 ESP32 內建 LED（GPIO2） | - |

### 類比 I/O

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **類比讀取** | 讀取類比值（0-4095） | 腳位編號（建議 GPIO32-39） |
| **PWM 輸出** | 輸出 PWM 訊號 | 腳位編號、值（0-255） |

### 時間與計時

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **等待** | 等待指定時間 | 時間（毫秒） |
| **經過時間** | 取得開機後經過的時間 | → 毫秒 |

### 序列通訊

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **序列開始** | 初始化序列通訊 | 波特率（9600/57600/115200） |
| **序列輸出** | 輸出到序列監視器 | 輸出值 |
| **序列輸出行** | 輸出並換行 | 輸出值 |

### 其他

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **播放音調** | 以指定頻率輸出聲音 | 腳位編號、頻率（Hz） |

## 資料（陣列）

用於陣列操作的積木。

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **建立陣列** | 建立 1D/2D/3D 陣列 | 變數名稱、類型、維度、大小 |
| **設定陣列元素** | 設定陣列元素的值 | 變數名稱、索引、值 |
| **取得陣列元素** | 取得陣列元素 | 變數名稱、索引 → 值 |
| **取得陣列大小** | 取得元素數量 | 變數名稱 → 數字 |
| **陣列內容** | 定義初始值 | 元素數量、各元素的值 |

## 感測器

### 超音波感測器（HC-SR04）

距離測量感測器。

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **HC-SR04 初始化** | 初始化超音波感測器 | Trig 腳位、Echo 腳位 |
| **超音波距離（cm）** | 測量距離 | → 公分值 |

**接線範例：**
- Trig → GPIO18
- Echo → GPIO19
- VCC → 5V
- GND → GND

### RUS-04（RGB 超音波感測器）

內建 RGB LED 的超音波感測器，適用於機器人。

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **RUS-04 初始化** | 初始化感測器和 RGB LED | Trig 腳位、Echo 腳位、RGB 腳位、LED 數量 |
| **RUS-04 距離（cm）** | 測量距離 | → 公分值 |
| **RUS-04 RGB 雙眼** | 設定雙眼顏色 | 左眼顏色、右眼顏色 |
| **RUS-04 RGB 單眼** | 設定單眼顏色 | 左/右、顏色 |
| **RUS-04 RGB 亮度** | 設定 LED 亮度 | 0-255 |
| **RUS-04 RGB 關閉** | 關閉 LED | - |

### 溫濕度感測器（DHT11/DHT22）

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **DHT 初始化** | 初始化 DHT 感測器 | 類型（DHT11/DHT22）、腳位 |
| **DHT 溫度（°C）** | 取得溫度 | → 攝氏 |
| **DHT 濕度（%）** | 取得濕度 | → 百分比 |

### 觸控感測器

ESP32 內建電容式觸控感測器。

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **觸控感測器初始化** | 初始化觸控感測器 | 腳位（T0-T9）、閾值 |
| **是否觸碰？** | 偵測觸碰 | → true/false |
| **觸控感測器數值** | 取得原始值 | → 數字（越低 = 觸碰越強） |

**支援的腳位：** T0(GPIO4)、T2(GPIO2)、T3(GPIO15) 等

### 聲音感測器

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **聲音感測器初始化** | 初始化麥克風感測器 | ADC 腳位（GPIO32-39） |
| **聲音大小** | 取得音量等級 | → 0-4095 |
| **偵測到聲音？** | 偵測超過閾值的聲音 | 閾值 → true/false |

### 光感測器

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **光感測器初始化** | 初始化 CdS 感測器 | ADC 腳位（GPIO32-39） |
| **光線強度** | 取得亮度 | → 0-4095（越高 = 越亮） |

### 數位感測器

| 積木名稱 | 說明 | 偵測內容 |
|----------|------|----------|
| **按鈕** | 按鈕/開關 | 按壓狀態 |
| **PIR 運動感測器** | 紅外線運動感測器 | 移動 |
| **傾斜感測器** | 傾斜感測器 | 傾斜 |
| **震動感測器** | 震動/衝擊感測器 | 震動 |
| **磁力感測器** | 霍爾效應感測器 | 磁鐵 |
| **光遮斷器** | 紅外線遮斷感測器 | 遮斷 |
| **紅外線障礙感測器** | 紅外線障礙感測器 | 障礙物 |
| **火焰感測器** | 火焰偵測感測器 | 火焰 |
| **氣體感測器** | MQ 系列氣體感測器 | 氣體 |
| **限位開關** | 微動開關 | 接觸 |

### 類比感測器

| 積木名稱 | 說明 | 輸出模式 |
|----------|------|----------|
| **可變電阻** | 可變電阻 | 原始值/百分比/角度 |
| **光感測器（LDR）** | CdS 光感測器 | 原始值/百分比 |
| **熱敏電阻溫度** | NTC 熱敏電阻 | 攝氏 |
| **LM35 溫度** | LM35 溫度感測器 | 攝氏 |
| **土壤濕度** | 土壤濕度感測器 | 原始值/百分比 |
| **水位感測器** | 水位感測器 | 原始值/百分比 |
| **搖桿** | 2軸類比搖桿 | X軸/Y軸 |

### 循跡感測器

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **通用循跡感測器初始化** | 初始化 QTR/TCRT 感測器 | 感測器數量（2-8）、各腳位 |

## 致動器

### 伺服馬達

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **伺服連接** | 將伺服連接到腳位 | 腳位編號 |
| **伺服角度** | 將伺服旋轉到角度 | 腳位、角度（0-180°） |
| **伺服角度（變數）** | 從變數設定角度 | 腳位、角度值 |
| **伺服斷開** | 斷開伺服（省電） | 腳位 |
| **伺服掃描** | 從起始角度移動到結束角度 | 腳位、起始角度、結束角度、速度 |

### 直流馬達（L298N）

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **馬達初始化** | 初始化 L298N 馬達驅動器 | 馬達（A/B）、IN1/IN2/ENA 腳位 |
| **馬達動作** | 驅動馬達 | 馬達、方向（前進/後退/停止）、速度（0-255） |
| **馬達速度** | 只改變速度 | 馬達、速度 |
| **馬達停止** | 停止馬達 | 馬達（A/B/兩者） |

### 步進馬達

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **步進馬達初始化** | 初始化 28BYJ-48 | IN1-IN4 腳位 |
| **步進馬達旋轉** | 按步數旋轉 | 步數、方向、速度 |

## NeoPixel（LED）

控制 WS2812B 等可定址 LED。

### 基本操作

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **NeoPixel 環初始化** | 初始化 LED 燈條 | 腳位、LED 數量 |
| **設定 LED 顏色** | 設定單個 LED 顏色 | 索引、R/G/B（0-255） |
| **設定所有 LED 顏色** | 設定所有 LED 相同顏色 | R/G/B（0-255） |
| **NeoPixel 顯示** | 套用顏色 | - |
| **NeoPixel 清除** | 關閉所有 LED | - |
| **NeoPixel 亮度** | 設定整體亮度 | 0-255 |

### 效果

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **彩虹效果** | 彩虹動畫 | 速度 |
| **彈跳效果** | 彈跳 LED | 顏色、速度 |
| **循環效果** | 旋轉效果 | 顏色、速度 |

### 簡易操作

| 積木名稱 | 說明 | 參數 |
|----------|------|------|
| **所有 LED 顏色（簡易）** | 按顏色名稱設定所有 LED | 顏色（紅/綠/藍/黃/紫/青/白/關） |
| **LED # 顏色（簡易）** | 按顏色名稱設定單個 LED | 索引、顏色 |
