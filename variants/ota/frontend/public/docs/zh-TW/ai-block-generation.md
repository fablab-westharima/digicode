# 使用 AI 積木生成

<!-- 此文件由 AI 翻譯自日文原版（2026-04-22） -->

**最後更新:** 2026-04-22

DigiCode 的 AI 積木生成功能，讓您可以用自然語言的提示自動生成 Blockly 積木。

---

## 1. 概覽

此功能採用 BYOK（自帶金鑰）方式，使用您自己的 AI API 金鑰進行生成。

| 項目 | 說明 |
|------|------|
| **方式** | BYOK（使用您自己的 API 金鑰） |
| **支援提供商** | OpenAI / Claude (Anthropic) / Google Gemini / 自訂（OpenAI 相容） |
| **可用方案** | Lite / Pro / Enterprise（student 帳戶除外） |
| **API 金鑰儲存** | 瀏覽器 localStorage（僅限本機裝置） |
| **傳送至 DigiCo** | 無（提示詞與生成結果直接從瀏覽器傳送至提供商） |

> **注意:** API 金鑰以明文儲存於瀏覽器 localStorage。與他人共用裝置時請格外謹慎。

---

## 2. 初始設定

1. 開啟側邊欄的帳戶選單
2. 點選 **「AI 設定」**（顯示在「變更密碼」正下方）
   - 僅 Lite / Pro / Enterprise 方案可見
3. 選擇提供商（OpenAI / Claude / Gemini / 自訂）
4. 輸入所選提供商的 API 金鑰
   - 金鑰取得方式請參閱各提供商官方網站（對話框內有連結）
5. 輸入模型名稱（省略則使用預設模型）
6. 點選 **「儲存」**

> **刪除 API 金鑰:** 若已輸入 API 金鑰，對話框中會顯示「刪除 API 金鑰」連結。點擊一次即可刪除並儲存。

---

## 3. 使用方式

1. 在編輯器頁面側邊欄底部找到 **「AI 積木生成」** 小工具
2. 在輸入欄中輸入提示詞（例：`讓第 13 腳位的 LED 每秒閃爍一次`）
3. 點選 **「生成」** 按鈕，或按 `Ctrl+Enter`（Mac：`Cmd+Enter`）
4. 生成的積木將新增至工作區

> **提示:** 生成的積木會**附加**到現有積木（不會刪除既有積木）。若想重新開始，請先手動清除工作區再生成。

## 5. 提示詞撰寫技巧

**越具體的提示詞，生成精度越高。**

| 模糊 | 具體 |
|---|---|
| 「傳送溫度感測器數值」 | 「用 MQTT 傳送 DHT11 的溫度與濕度」 |
| 「往前移動」 | 「前進 2 秒後停止」 |
| 「量測距離」 | 「用 HC-SR04 超音波感測器量測距離並輸出至序列埠」 |

- **使用模式詞彙**: `robots_wheel` 模式請用「前進」、「右轉」；`homeassistant` 模式請用「MQTT 發佈」、「WiFi 連線」
- **支援多語言輸入**: 提示詞可用中文、英文、日文、西班牙文等多種語言

## 6. 常見問題排除

| 症狀 | 原因與解決方式 |
|---|---|
| **網路錯誤（可能為 CORS）** | API 金鑰可能無效或已過期。請在 AI 設定中重新輸入有效金鑰 |
| **429 錯誤（速率限制）** | 已達到請求上限或免費額度已用盡。請至提供商後台確認 |
| **404 錯誤（找不到模型）** | 模型名稱可能已廢棄或變更。請參閱提供商官方文件確認有效的模型名稱 |
| **生成後未出現積木** | 3 次重試後仍無法生成有效積木。請重新撰寫更具體的提示詞 |
| **「未設定 API 金鑰」錯誤** | 請先在 AI 設定中輸入並儲存 API 金鑰 |

## 7. 已知限制（MVP 階段）

目前的 AI 積木生成為初始版本（MVP），以下限制將於後續更新中改善。

- **僅限附加模式**: 生成的積木會附加至現有積木（非取代）。若要重新開始，請手動清除工作區
- **機器人模式不含 WiFi/BLE 積木**: `robots_wheel` / `robots_humanoid` / `robots_transform` 模式不包含 WiFi 或 BLE 類別。若需結合 WiFi/BLE，請切換至 `all_blocks` 或 `generic` 模式再生成
- **單次生成**: 每次提示詞只生成一次。AI 不會記住上次的生成內容（對話模式尚未實作）

---

## 8. 隱私與安全

- **API 金鑰儲存**: 以明文儲存於瀏覽器 localStorage，存在 XSS 洩漏風險。不建議在與他人共用的裝置上使用此功能
- **資料傳輸**: 提示詞與生成結果直接從瀏覽器傳送至 AI 提供商，不經過 DigiCo LLC 的伺服器
- **計費**: 使用費用直接計入您的提供商帳戶，DigiCo LLC 不參與任何計費事宜
